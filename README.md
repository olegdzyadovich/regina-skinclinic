## Stack

Static site, no build step.

- `index.html` — semantic HTML
- `styles.css` — design tokens, layout, typography (Cormorant Garamond + Inter)
- `script.js` — reveal-on-scroll, mobile menu, contact form ack

## Local preview

```sh
open index.html
```

Or any static server:

```sh
python3 -m http.server 8000
# → http://localhost:8000
```
